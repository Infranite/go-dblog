# 安全策略

[English](./SECURITY.md)

## 支持版本

安全修复面向当前公开 tag 集合 `v1.0.0` 和默认分支。后续版本的受支持版本窗口记录在
GitHub Releases 中。

## 报告漏洞

请使用 GitHub private vulnerability reporting，或为仓库创建 private security
advisory。修复可用前，不要在公开 issue 中发布利用细节。

报告中请包含：

- 受影响的 module 和 version 或 commit；
- 复现问题所需的最小输入；
- 预期影响，例如 panic、资源耗尽、不安全闪回输出或错误事件解码。
